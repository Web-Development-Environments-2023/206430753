❔ Answers To Questions

1. What are the main diffrences between web 1.0, web 2.0 and web 3.0?

The main diffrence between web 1.0 to web 2.0 and web 3.0 is that in web 1.0 the websites were static. A website was created and other users couldnt change it or interact with it, they could only read it. In web 2.0 and web 3.0 that changed. The websites became interactive, anyone can edit and share information. In addition the search engines in web 1.0 were very diffrent. The search in web 1.0 was based on key words. The results were returned based on the searched text and not based on relevance. In web 2.0 and web 3.0 that was devedloped and users could search a question and get links to websites with the most relevence to the quary they presented. The main diffrence between web 2.0 and web 3.0 is that in web 3.0 the Internet became a database of knowlege. The websites are the content of the database and becase of that the data that we can search and see is more procceced, and the web can present the most relevant content to each user in a personalized way.

2. Explain the diffrences above with the help of a use case.

To explain diffrence between websites in web 1.0 and websites in web 2.0 and web 3.0 we will talk about a blog. In web 1.0 a person created a blog and uploaded articles. other users of the web coulf only observe the content and nothing else. In web 2.0 the blog could be interactive with other users. The users coulg live comments on diffrent articles in the blog and create a discussion between users of the web.
An example to the diffrence between the search engines in web 1.0 to web 2.0 and web 3.0 we will talk about a query. If we searched the query "The presidential campaign" we would get various results. Because the search is based on key words websites with the word "presidential" or "campaign" even not relates to the combination of the two would come up. Contrary to that in web 2.0 and 3.0 the search engines take in to account the meaning of the whole sentence and the combiation of words. So websites about food campaigns for example wouldnt come up as results.
A big diffrence between web 2.0 and web 3.0 is the ability to learn the users. In web 2.0 if a user searched a product to buy the results would be a list of websites with the product listed based on the popularity of the website, advertising of the website etc. In web 3.0 the web is learning the prefrences of every user. If you searched a product before, bought from a website before or even clicked on a link to some product, the web remenbers it and recommands to you the products that are similar to what you have seen before.
